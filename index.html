Comandos en Terminal:

pwd = ver ruta actual

cd = Change directory

cd .. = retroceder a la carpeta anterior

cd / = Directorio Raiz

cd mnt = Ir a Windows

cd ~ = Ir a home Linux

ls = ver el contenido de la capeta donde te encuentras

clear = Limpiar Terminal

sudo = Crear carpeta con permisos de administrador

mkdir = crear carpeta

touch = crear archivo

mv = mover

code . = Abrir VS Code en la carpeta actual

cat {nombre del archivo + extencion} = ver el contenido del archivo

man {comando} = Manual del comando {q = para salir}

Instalar Aplicaciones y Actualizaciones

sudo apt-get update = (ver actualizaciones)

(permiso) (instalar)

sudo apt-get upgrade = (Actualizar)

sudo apt install nodejs (nombre del programa)

node -v = ver version del programa

sudo apt intall npm = node package manager (administrador de paquetes de node)

npn = instalar paquetes de forma permante

npx = ejecutar paquetes desde internet sin instalar

rm {archivo.extencion} = Borrar archivo

rm -d {nombre carpeta} = Borrar carpeta vacia

rm -rf {nombre carpeta} = Borrar carpeta
